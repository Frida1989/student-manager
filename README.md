#  Student Manager CRUD App

A frontend application built with HTML, CSS, and JavaScript for managing student data.  
The app demonstrates full CRUD functionality using the Fetch API and includes UI features like modal editing, sorting, and persistent view settings with localStorage.

---

##  Features

 Fetch students from API (GET)
   Add new students (POST)
   Edit students in a modal (PUT)
   Delete students (DELETE)
   Sort students by first name
   Toggle between card view and list view
   Save selected view in localStorage (persists after reload)
  Clear input fields after adding a student

---

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- LocalStorage
- JSON Server (for local API)
